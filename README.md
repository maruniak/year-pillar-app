# Year Pillar App — Шаг 1 (Годовой столп)

Статическое приложение считает **только годовой столп** по дате рождения с возможностью задавать **дату начала китайского года** (Ли Чунь / Китайский Новый год / Произвольная дата).

## Быстрый старт (GitHub Pages, без сборки)
1. Создайте новый публичный репозиторий, напр., `year-pillar-app`.
2. Скопируйте содержимое этой папки в репозиторий **как есть** (включая каталог `docs/`).
3. В настройках репозитория: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, затем **Branch: `main` / Folder: `/docs`**, нажмите **Save**.
4. Через ~1 минуту ваш сайт будет доступен по адресу:  
   `https://<ваш_аккаунт>.github.io/<имя_репозитория>/`

> Почему `/docs`: GitHub Pages умеет публиковать статические файлы прямо из папки `/docs` без GitHub Actions.

## Локальный запуск (любой статический сервер)
Откройте `docs/index.html` в браузере или поднимите простой сервер, например Python:
```bash
cd docs
python3 -m http.server 8080
```
Откройте http://localhost:8080

## Функционал (Шаг 1)
- Ввод: дата рождения `YYYY-MM-DD`, режим границы года (Ли Чунь / КНГ / Произвольная).
- Расчёт: годовой **Gan-Zhi** с учётом границы (если `birthDate < boundaryDate` ⇒ берём год-1).
- Вывод: иероглифы ствола/ветви, Инь/Ян, элемент, животное, № в 60-летнем цикле, TCM-пара органов, JSON для копирования.
- История: последние 5 расчётов (localStorage).
